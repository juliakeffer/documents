Extracting File and Directory Contents

You can extract individual files and entire directories of files. You may want to extract files you tagged, files that Autopsy identified during ingest, such as Microsoft Offices files, or other files in the disk image. For example, you might want to extract the contents of the Documents and Settings directory from one of the user directories.

To extract file and directory contents

In the Data Explorer, click the folder that contains the file or directory you want to extract. For example, click Views to see the analyzed files.

Click the subfolders until you see the folder contents in the Result Viewer. In the example below, to see the list of Microsoft Office files in the Result Viewer, click the folders View > File Types > Documents > Office.

In the Result Viewer, right-click the file or directory that you want to extract and then click Extract File.

A new window opens to enable you to save the file.

If you want to save the file in a different directory, change to the directory where you want to store the file.

If you want to change the file name, in the File name box, type the file name you want to use.

Click Save. The Information dialog box appears.

To close the box, click OK.

You have finished extracting the file.
