Creating a Basic Report

A basic report uses the default data selections. You need to select the format you want to use for the report (HTML or Excel).

On the toolbar, click Generate Report. The Generate Report window appears.

To create an HTML report, select the HTML check box. To create an Excel report, select the Excel check box.

You can create more than one report format at the same time.

Click Next. The Generate Report window updates to show Configure HTML and Excel reports.

By default, Autopsy reports all results.

To create a basic report, click Finish. The Report Generation Progress window appears.

Depending on the size of the report, it may take Autopsy a few seconds to generate the report. The report is ready to view when you see the check mark beside the progress bar. Autopsy shows links for the reports you generated.

When the report finishes, click the link for the report type you want to view.

To view the HTML report, click the link beside HTML in the window. Your default browser opens and shows the report.

To view an Excel report, click the link beside Excel in the window. Microsoft Excel opens and shows the report.

To close the Report Generation Progress window, click Close.

If you want to look at a report later, you can find it in the Reports subdirectory of the case directory.
