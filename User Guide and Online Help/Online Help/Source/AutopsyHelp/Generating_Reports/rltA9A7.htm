Creating a Custom Report

You can create two kinds of custom reports

·       A report that shows a list of files you tagged

·       A report that shows only the data type you select, for example, a report showing all Internet artifacts

To create a report showing tagged files

1.   On the toolbar, click Generate Report. The Generate Report window appears.

2.   Select the HTML check box to create an HTML report and select the Excel check box if you want to create an Excel report. Click Next. The Generate Report window updates to show Configure HTML and Excel reports.

3.   To create a report that shows a list of files you tagged, click Tagged Results. The tag names become available.

4.   Select the tag name check box for the tags you want to report. Click Finish. The Report Generation Progress window appears.

Depending on the size of the report, it may take Autopsy a few seconds to generate the report. The report is ready to view when you see the check mark beside the progress bar. Autopsy shows links for the reports you generated.

5.   When the report finishes, click the link for the report type you want to view.

♦      To view the HTML Hyper-Text Markup Language report, click the link beside HTML in the window. Your default browser opens and shows the report.

♦      To view an Excel report, click the link beside Excel in the window. Microsoft Excel opens and shows the report.

6.   To view the HTML Hyper-Text Markup Language report, click the link beside HTML in the window. Your default browser opens and shows the report. To view an Excel report, click the link beside Excel in the window. Microsoft Excel opens and shows the report.

7.   To close the Report Generation Progress window, click Finish.

To create a report with selected categories of data

1.   Click Generate Report on the toolbar. The Generate Report window appears.

2.   Select the HTML check box to create an HTML report and select the Excel check box if you want to create an Excel report. Click Next. The Generate Report window updates to show Configure HTML and Excel reports.

3.   Click Advanced. The Advanced Artifact Selection window appears.

4.   Select the check box for the items you want to view in your report. To return to the Configure HTML and Excel reports window, click OK .

5.   Click Finish. The Report Generation Progress window appears.

Depending on the size of the report, it may take Autopsy a few seconds to generate the report. The report is ready to view when you see the check mark beside the progress bar. Autopsy shows links for the reports you generated.

6.   When the report finishes, click the link for the report type you want to view.

♦      To view the HTML Hyper-Text Markup Language report, click the link beside HTML in the window. Your default browser opens and shows the report.

♦      To view an Excel report, click the link beside Excel in the window. Microsoft Excel opens and shows the report.

7.   To close the  Report Generation Progress window, click Finish.
