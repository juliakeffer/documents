Creating a Custom Report

You can create two kinds of custom reports

·       A report that shows a list of files you tagged

·       A report that shows only the data type you select, for example, a report showing all Internet activity

To create a report showing tagged files

1.   On the toolbar, Click Generate Report. The Generate Report window appears.

2.    Select the report types you want to create and click Next. The Generate Report window updates to show Configure HTML Hyper-Text Markup Language and Excel reports panel.

3.   To create a report that shows a list of files you tagged, click Tagged Results. Select the check box for the tags you want to report. Click Finish. The Report Generation Progress window appears.

4.   When the report finishes, click the link for the report type you want to view. The following shows an example of a tagged results report in HTML Hyper-Text Markup Language format.

To create a report with selected categories of data

1.   Click Generate Report on the toolbar. The Generate Report window appears.

2.   Select the report types you want to create and click Next. The Generate Report window updates to show Configure HTML Hyper-Text Markup Language and Excel reports panel.

3.   Click Advanced. The Advanced Artifact Selection window appears.

4.   Select the items you want to view in your report and click OK to return to the Configure HTML Hyper-Text Markup Language and Excel reports window.

5.   Click Finish. The Report Generation Progress window appears.

6.   When the report finishes, click the link for the report type you want to view.
