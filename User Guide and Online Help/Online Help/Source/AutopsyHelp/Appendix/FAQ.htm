Frequently Asked Questions

This section provides answers to common user questions.

What disk image formats can I use?

You can use raw disk images, which can be single or multiple files, and E01 (EnCase) format.

What file systems can Autopsy analyze?

NTFS

HFS+

ISO9660

 Ext2, Ext3

FAT12, FAT16, FAT32

UFS

How do I remove a disk image from a case?

You cannot remove a disk image from a case. You need to create a new case and ensure that you only add the necessary images.

Why is the timeline information empty?

Timeline features are only supported on Windows file system disk formats (FAT, NTFS).
