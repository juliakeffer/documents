Cancelling and Restarting Ingest

If ingest takes too long, you can cancel it and reduce the set of data to analyze.

To cancel ingest

In the bottom right corner of the main window, right-click the blue progress bar and then click Cancel Process.

The Cancel Running Task box appears.

To stop ingest, click Yes. Ingest stops and the blue progress bar at the bottom of the window disappears.

To restart ingest

You can change the set of ingest modules to use and start ingest again.

Click the Images section of the Data Explorer. Right-click the image you want to process and then click Restart Ingest Modules.

The Ingest Modules window appears.

Select the module name check boxes for the modules you want to use and click Start.

Ingest starts and you can monitor the progress in the bottom right corner of the window.
