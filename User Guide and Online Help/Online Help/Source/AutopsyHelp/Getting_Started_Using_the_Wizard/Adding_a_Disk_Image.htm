Adding a Disk Image

The first time you create a case, you will automatically see the Add Image window, which takes you through the steps to add an image to the case.

Once you add an image to a case, you cannot remove it.

The Add Image window enables you to select the image.

To add a disk  image to a case

In the Add Image window, click Browse and navigate to the directory where the image is stored. Click the name of the image file and click Open.

If the investigator collected the image in a different time zone, in the time zone list, click the appropriate time zone.

Click Next. The Add Image window updates to show the Configure Ingest Modules wizard.

You are now ready to configure disk analysis, which Autopsy calls ingest. The wizard takes you to the next step automatically.
