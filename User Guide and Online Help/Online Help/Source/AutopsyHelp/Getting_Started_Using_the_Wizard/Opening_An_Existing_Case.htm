Opening an Existing Case

If you previously created a case, you can open it again either by navigating to it, or Autopsy can show you a list of previously open cases to choose from. If you currently have a case open, on the toolbar, click Close Case to close it.

To open an existing case

In the Welcome window, click Open Existing Case. The Open window appears.

Navigate to the directory where you stored the case.

Click the file name of the case. It has the file extension .aut.

Click Open. The case opens in Autopsy.

To open a recent case

In the Welcome window, click Open Recent Case. The Open Recent Case window shows a list of recent cases you have worked on.

Click the name of the case you want to open and click Open. The case opens in Autopsy.
