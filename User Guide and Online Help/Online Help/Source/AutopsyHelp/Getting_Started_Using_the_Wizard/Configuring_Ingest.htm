Configuring Ingest

In the previous step, you added the image and the Add Image window showed the Configure Ingest Modules wizard.

To configure ingest

Select the module name check boxes for the modules you want to use to analyze the disk image contents.

Click Next. The Add Image window changes to show Adding Image – Complete. The image is part of the case and the ingest process begins.

To close the wizard, click Finish.

You can monitor the progress in the bottom right corner of the main window.

To view progress messages, click the  icon on the toolbar at the top of the window. You can begin to explore the analyzed data while ingest is in progress.
