Exploring Analyzed Data

The Autopsy interface window has three regions: the Data Explorer Central area of main window shows file lists (left panel), the Result Viewer (main window), and the Content Viewer (bottom panel).

Using the Data Explorer

As ingest The process of extracting and classifying information from the image file added to Autopsy. runs, it categorizes information. The Data Explorer Central area of main window shows file lists shows the results in a tree view.

·       The Images section shows the image contents in Windows Explorer format.

·       The Views section shows the files that Autopsy could categorize based on the file metadata Properties of a file attached to the file by some programs.

·       The Results section shows the results of the data extracted during the ingest The process of extracting and classifying information from the image file added to Autopsy. process.

·       The Keyword Hits section shows search results from custom or pre-defined search criteria.

If you click any item in the tree, the Result Viewer shows the contents. You can click individual items in the Result Viewer to view the contents in the Content Viewer Area of the main window to view file contents.

Using the Result Viewer

The Result Viewer shows the results of selecting an item in the Data Explorer panel. You can display the results in different formats, depending on the type of file. Figure 4: Result Viewer Directory View shows a list of files categorized as Microsoft Office file types.

The Directory Listing displays the data as a table with the properties of each file. The properties shown are: name, time (modified, changed, accessed, and created), size, flags (directory and metadata Properties of a file attached to the file by some programs), mode, user ID, group ID, metadata Properties of a file attached to the file by some programs address, attribute address, and type (directory and meta). Click the Table View tab to see this view.

If you view a list of images, in addition to the Table View, you can use the Thumbnail View tab to see the images, instead of a list of files. Autopsy supports viewing JPG Graphics file type/extension (lossy compressed 24 bit color image storage format developed by the Joint Photographic Experts Group), PNG Left panel of main window, and GIF Graphic Interchange Format image formats. See Figure 5: Result Viewer Thumbnail View.

If you perform a keyword search, you can see the results by clicking the Keyword search tab. See Figure 6: Result Viewer Keyword Search.

The Keyword search tab separates the list into keyword matches for each keyword list. Once you close the Keyword search tab, you can view the list by clicking the list name under the Keyword Hits section of the Data Explorer Central area of main window shows file lists panel.

When you click a file in any view of the Result Viewer, the contents appear in the Content Viewer.
