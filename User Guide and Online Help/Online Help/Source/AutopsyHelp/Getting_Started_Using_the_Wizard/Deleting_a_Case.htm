Deleting a Case

If you do not need a case any more, you can either delete it from your hard drive in Windows Explorer, or you can delete it using Autopsy.

To delete a case using Autopsy

On the File menu, click Case Properties.

The Case Properties window opens and shows the case information.

In the Case Properties window, click Delete Case. The Warning: window appears.

To delete the case, click Yes.

You have successfully deleted the case.
