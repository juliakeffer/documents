Saving File Locations

After you run a keyword search, you may want to save the location of a file with a keyword match. You can save the file locations of any file you see in the Result Viewer. Autopsy uses tags, which are similar to bookmark folders in a bookmark list in a web browser. After you create a tag, you assign the file to the tag. Tagging the file is similar adding a bookmark to a web page a a web browser's bookmark folder. Before you tag a file, you need to create a tag list. After you create a tag, you may want to create a report containing the list of tagged files.

To create a tag list

In the Result Viewer, click the Keyword search tab.

Right-click the file you want to tag, point to Tag File and then click Create a new tag.

The Create a new tag window opens.

In the Tag Name box, type a descriptive name for the tag list. Click OK.

The tag name appears under the Tags area in the Data Explorer.

To tag a file

In the Result Viewer, right-click the file you want to tag, point to Tag Source File and then click the name of the tag you want to apply to the file. In the example below, the tag Passwords is applied to the file.

To see the list of files you marked, under the Tags area in the Data Explorer, click the tag name.

The list of tagged files and their locations appears in the Result Viewer.

You have now saved the location of the file for later use.
