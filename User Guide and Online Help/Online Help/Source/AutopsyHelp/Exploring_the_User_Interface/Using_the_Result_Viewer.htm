Using the Result Viewer

When you select an item in the Data Explorer, the Result Viewer shows the results. You can view the results in different formats, depending on the type of file. In the following example, the Result Viewer shows a list of files categorized as Microsoft Word file types.

You can use the Table View tab under the Directory Listing tab to display the data as a table with the properties of each file. The properties shown are

name

time (modified, changed, accessed, and created)

size

flags (directory and metadata)

mode

user ID

group ID

metadata address

attribute address

type (directory and metadata)

If you view a list of images, in addition to the Table View tab, you can use the Thumbnail View tab to see the images, instead of a list of files.

Autopsy supports viewing JPG, PNG, and GIF image formats.

If you perform a keyword search, to see the results, click the Keyword search tab.

The Keyword search tab separates the list into keyword matches for each keyword list. Once you close the Keyword search tab, to view the list again, click the list name under the Keyword Hits section of the Data Explorer.

When you click a file in any view of the Result Viewer, the contents appear in the Content Viewer.
